# PlaylistParser
EXTM3U playlist parser

Написано для работы с плейлистами твича.

TargetFramework net6.0

## Как использовать

MasterParser и MediaParser для мастер и медиа плейлистов соответственно.

Метод Parse вернёт модель плейлиста. Всио.
